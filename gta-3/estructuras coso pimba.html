📊 Comparación — Steam vs Twitter

  • Similitudes:
    • Ambas usan header + nav + main + footer
    • Ambas tienen aside (sidebar) con contenido secundario
    • Ambas usan div para agrupar elementos de layout
    • Ambas usan span para destacar texto inline (badges, nombres)
  • Diferencias:
    • Steam: nav horizontal arriba como barra
    • Twitter: nav vertical a la izquierda como columna
    • Steam: contenido en grillas de tarjetas (div.card)
    • Twitter: contenido en lista vertical de artículos (tweets)
    • Steam: búsqueda en el header
    • Twitter: formulario de publicación dentro del main
  • Steam — coincidencia estimada con estructura real: ~75%
  • Twitter — coincidencia estimada con estructura real: ~80%
